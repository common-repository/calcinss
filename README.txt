== Description ==

Este plugin faz o cálculo do ano de aposentadoria seguindo a regra 85/95, que calcula o tempo de contribuição e idade para descobrir o ano de aposentadoria integral pelo INSS.as.

== Installation ==

Como instalar o Plugin CalcINSS

1. Envie os arquivos da pastas 'wpCalcFinanciamento` para o diretório `/wp-content/plugins/`;
2. Ative o plugin;
3. Copie e Cole o shortcode `[CalcINSS]` em sua página ou post.
